Book Cover Redesign

  • Medium: Paper 8.5in x 5.5in
  • Date: Fall 2022
  • Programs Used: Illustrator, Photoshop, Indesign

The books presented here are were all written by or about West Virginians. To modernize these titles, each entry's cover was redesigned to be presented as a set. To further this implication, a fictional collection for the titles to be included in was created as well. For these redesigns to feel realistic, they each include bar codes, accurrately measured dimensions based on page count, and a publishing logo as well.

As the content in each book is about toys or toy companies, a primary goal for this project was to use type that reflected the fun nature of toys. Thick sans serifs paired with a relaxing serif to scan the book's summary create a visually pleasing display for each work. Utilizing lines and patterns are also an important element which ties the pieces together as well.
